# 🚀 快速部署指南（5分钟搞定）

> ⚠️ **如果遇到网络连接问题**（如 `ENOTFOUND api.vercel.com`），请查看 [网络问题解决方案.md](./网络问题解决方案.md)

## 最简单的方法：使用 Railway（推荐，国内稳定）⭐

**Railway 在国内访问更稳定，强烈推荐！**

### 步骤1：访问Railway
- 打开 https://railway.app
- 使用GitHub账号登录

### 步骤2：创建项目
- 点击 "New Project"
- 选择 "Deploy from GitHub repo"
- 如果没有连接GitHub，先点击 "Connect GitHub" 授权

### 步骤3：选择仓库
- 如果代码已在GitHub，直接选择你的仓库
- 如果代码在本地，先推送到GitHub：
  ```bash
  git init
  git add .
  git commit -m "Initial commit"
  git remote add origin https://github.com/你的用户名/你的仓库名.git
  git push -u origin main
  ```

### 步骤4：自动部署
- Railway会自动检测Node.js项目
- 自动安装依赖并启动服务
- 等待部署完成（约1-2分钟）

### 步骤5：获取URL
- 部署完成后，点击项目
- 在 "Settings" -> "Networking" 中可以看到公网URL
- 或点击 "Generate Domain" 生成域名

### 步骤6：完成！
部署完成后会显示一个URL，例如：
```
https://your-project.up.railway.app
```

这个URL就是你的服务地址，**24小时在线，电脑关机也能访问！**

**优点**：
- ✅ 国内访问稳定
- ✅ GitHub推送自动部署
- ✅ 支持文件写入（Vercel不支持）

## 备选方案：使用 Vercel（如果网络正常）

### 方法A：使用Vercel网页版（推荐，无需命令行）

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 使用GitHub账号登录

2. **导入项目**
   - 点击右上角 "Add New..." -> "Project"
   - 选择 "Import Git Repository"
   - 如果没有看到仓库，点击 "Adjust GitHub App Permissions" 授权

3. **配置项目**
   - 选择你的代码仓库
   - Framework Preset: 选择 **"Other"**
   - Install Command: `npm install`
   - **重要**：在 "Override" 中设置 Start Command: `node server.js`

4. **部署**
   - 点击 "Deploy"
   - 等待部署完成

### 方法B：使用Vercel CLI（需要网络正常）

```bash
# 安装
npm install -g vercel

# 登录
vercel login

# 部署
vercel
```

### 完成！
部署完成后会显示一个URL，例如：
```
https://list-manager-server.vercel.app
```

这个URL就是你的服务地址，**24小时在线，电脑关机也能访问！**

### 测试
```bash
# 获取list
curl https://你的项目名.vercel.app/api/list

# 添加元素
curl -X POST https://你的项目名.vercel.app/api/list \
  -H "Content-Type: application/json" \
  -d '{"item":"测试"}'
```

## 其他平台（同样简单）

### Render（稳定可靠）

1. 访问 https://render.com
2. 用GitHub登录
3. 点击 "New" -> "Web Service"
4. 选择仓库，配置：
   - Build Command: `npm install`
   - Start Command: `npm start`
5. 点击 "Create Web Service"
6. 等待部署完成

**优点**：国内访问稳定，免费套餐可用

## 💡 重要提示

1. **数据存储**：如果使用Vercel，文件写入可能有限制。建议使用Railway，它支持文件写入。

2. **更新代码**：
   - Vercel: 运行 `vercel` 命令
   - Railway: 推送代码到GitHub，自动部署

3. **查看日志**：各平台都提供日志查看功能，方便调试
